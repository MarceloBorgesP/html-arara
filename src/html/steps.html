HTML Arara provides a stepper. It conveys progress through numbered steps and may also be used for navigation.

Customize your Stepper

By default, the stepper has 4 steps, in case you want more or less steps you just need to create or delete a step: 

  
      <li class="step">
        <a href="#" class="mdl-button mdl-js-button" >
          <span>5</span>
          <h5>New Step</h5>
        </a>
      </li>

And add a new css rule: 

  
    .step {
      width: 20%
    }

The new rule must correspond to the number of steps you want. Have in mind that we are thinking in a scale of 100%. So if you want 5 steps, each step should have 20% of the total width.
